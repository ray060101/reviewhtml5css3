/*

前面是一些JSX的部分，主要實現功能與互動（onChange）在這邊都是儲存，可以先忽略。
第一部分是form製作一個表單，用到的技術有以下：
輸入元素：
    * <input>：用來接收使用者輸入的值。
        * type 屬性：設定輸入元素的類型（例如：text、tel、email、date 等）。
            * text：用於接收使用者輸入文字或字串的值。
            * tel：用於接收使用者輸入電話號碼的值，且會自動進行格式驗證。
            * email：用於接收使用者輸入電子郵件地址的值，且會自動進行格式驗證。
            * date：用於接收使用者輸入日期的值，且會提供日期選擇器的功能。
        * value 屬性：設定輸入元素的值。
        * onChange 屬性：設定當使用者輸入值時的處理函式。
        * pattern 屬性：設定輸入值的格式。
        * required 屬性：設定是否為必填欄位。
        * placeholder 屬性：設定在輸入框中顯示提示文字，一般會用較淺色的文字表示，例如"請輸入您的姓名"等等。
        * min 屬性：設定輸入值的最小值（僅適用於 type="range"）。
        * max 屬性：設定輸入值的最大值（僅適用於 type="range"）。
選擇元素：
    * <select>：用來讓使用者選擇一個選項。
        * value 屬性：設定選擇元素的值。
        * onChange 屬性：設定當使用者選擇選項時的處理函式。
    * <option>：用來定義選擇元素的選項。
        * - value 屬性：設定選項的值。
按鈕元素：
    * <input>：用來提交表單。
        * type 屬性：設定按鈕的類型（例如：submit、button 等）。
        * value 屬性：設定按鈕的顯示文字。
標籤元素：
    * <label>：用來為表單元素增加文字標籤。
表單元素：
    * <form>：用來包裹整個表單。
*/
